# Data Analytics Project 2 – Exploratory Data Analysis (EDA)

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on an E-Commerce Sales Dataset using Microsoft Excel.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Correlation Analysis
* Box Plot

## Analysis Performed

* Data Cleaning
* Descriptive Statistics
* Product Analysis
* Payment Method Analysis
* Referral Source Analysis
* Monthly Sales Trend Analysis
* Outlier Detection
* Correlation Analysis

## Key Findings

* Online payment was the most frequently used payment method.
* Credit Card generated the highest revenue.
* Instagram generated the highest number of orders.
* Email marketing generated the highest revenue.
* June recorded the highest monthly sales.
* UnitPrice showed the strongest positive correlation with TotalPrice.
